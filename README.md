# Bears plugin

`@Bears` is a compact tooling package.

## Contents
- `scripts/bears_auto_cd.py` — automatic CD executor.
- `assets/catalog/git-deploy-contract.v1.json` — Git target contract shape.
- `assets/catalog/cd-kube-deploy-contract.v1.json` — Kubernetes CD contract shape.
- `skills/mcp-designer/` — MCP surface design and review skill.

## Boundary
- Plugin skills are task procedures, not workspace instruction authority.
- Active workspace behavior is owned by `AGENTS.md` files and non-plugin contracts.
- No role prompts, hooks, workflows, generated inventories, or issue templates.
