# Bears plugin tooling package

This repository is a local tooling package. It is not an instruction source.

Authoritative agent behavior lives in active `AGENTS.md` files outside this plugin and in tracked non-plugin contracts selected by those routers.

This package may contain:

- executable helper scripts;
- disabled hook helper code not referenced by the manifest;
- schema/catalog data used by scripts;
- preserved migration docs that are not loaded as instructions.

This package must not expose:

- plugin `skills` discovery;
- `defaultPrompt` text;
- active `skills/*/SKILL.md` files;
- active role/profile prompts;
- environment-specific local paths as policy.

CD execution entrypoint: `scripts/bears_auto_cd.py`.

Repo-local deployment policy is resolved from the caller repo, for example `local_cd/policy/*.v1.json` in `BearsCLOUD/bears-infra`.
